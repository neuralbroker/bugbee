ชุดโครงการชุมชนที่สร้างขึ้นบน Bugbee

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ Bugbee ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-bugbee](https://github.com/awesome-bugbee/awesome-bugbee) และ [bugbee.cafe](https://bugbee.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
