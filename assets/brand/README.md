# Bugbee brand assets

- `bugbee-mark-v2.png` — dark primary mark for light backgrounds.
- `bugbee-mark-light.png` — light primary mark for dark backgrounds.
- `bugbee-github-social.png` — GitHub social-preview image (1280 × 640).

To use the social preview on GitHub, upload `bugbee-github-social.png` in the
repository's **Settings → General → Social preview** section.
